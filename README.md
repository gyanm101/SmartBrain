# SmartBrain

A full-stack face recognition web application that detects faces in uploaded images using AI, with secure authentication and real-time processing.

---

## Features
- Detect faces in images using Clarifai API  
- User authentication with JWT  
- Secure password storage using bcrypt  
- Real-time image processing  
- Responsive UI across devices  

---

## Tech Stack
- Frontend: React.js  
- Backend: Node.js, Express.js  
- Database: PostgreSQL  
- APIs: Clarifai Face Detection API  
- Deployment: Heroku  

---

## ▶️ Run the Project

## Frontend
```bash
cd frontend
npm install
npm run dev
```

## Backend
```bash
cd backend
npm install
npm start
```
